# CREDIT-CARD-FRAUD-RISK-ANALYSIS
A Power BI project that analyzes credit card transactions to detect and assess fraud risk using interactive dashboards and DAX measures. It visualizes fraud by location, time, and risk level, highlighting critical risk patterns and insights to support data-driven fraud prevention and financial security.
